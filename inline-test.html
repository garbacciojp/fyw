Demo 1: Basic Inline Embed

The widget is embedded inline with a fixed height container. Notice how the page remains scrollable and the widget stays within its container.

Implementation:

<!-- HTML --> <div id="inline-widget-1" style="height: 700px;"></div> <!-- Load Widget CSS and JS --> <link rel="stylesheet" href="http://fyw-lrqe8.ondigitalocean.app/widget/v2/style.css"> <script src="http://fyw-lrqe8.ondigitalocean.app/widget/v2/find-your-word-v2.iife.js"></script> <!-- Initialize Inline Mode --> <script> // Create new instance in inline mode const widget = new FindYourWordWidget({ mode: 'inline', container: '#inline-widget-1', debug: true }); </script>

Demo 2: Shopify Collection Page Integration

Example of how the widget would appear on a Shopify collection page, embedded among other content like product listings or category descriptions.

<!-- Shopify Liquid Template Example --> <div class="collection-content"> <!-- Collection Header --> <h1>{{ collection.title }}</h1> <p>{{ collection.description }}</p> <!-- Find Your Word Widget Inline --> <div class="find-your-word-section" style="margin: 40px 0; height: 700px;"> <div id="fyw-inline-widget"></div> </div> <!-- Product Grid Below --> <div class="product-grid"> {% for product in collection.products %} <!-- Product cards --> {% endfor %} </div> </div> <!-- Load Widget --> <link rel="stylesheet" href="https://fyw-lrqe8.ondigitalocean.app/widget/v2/style.css"> <script src="https://fyw-lrqe8.ondigitalocean.app/widget/v2/find-your-word-v2.iife.js"></script> <script> new FindYourWordWidget({ mode: 'inline', container: '#fyw-inline-widget' }); </script>
