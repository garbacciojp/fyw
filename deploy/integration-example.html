Find Your Word Widget - Integration Example

This page demonstrates how to integrate the Find Your Word widget into any website.

Method 1: Manual Control

Add a button that opens the widget when clicked:

<script src="https://your-domain.com/widget/v2/find-your-word-v2.iife.js"></script>
<button onclick="window.FindYourWordWidget.open()">
  Find Your Word
</button>

Method 2: Auto-Open

Widget opens automatically when page loads:

<script 
  src="https://your-domain.com/widget/v2/find-your-word-v2.iife.js"
  data-auto-open="true"
></script>

Method 3: Debug Mode

Enable debug mode for development:

<script 
  src="https://your-domain.com/widget/v2/find-your-word-v2.iife.js"
  data-debug="true"
></script>

API Reference

The widget exposes the following global API:

  • window.FindYourWordWidget.open() - Open the widget
  • window.FindYourWordWidget.close() - Close the widget
  • window.FindYourWordWidget.isOpen() - Check if widget is open
